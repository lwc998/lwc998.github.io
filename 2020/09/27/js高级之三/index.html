函数的定义和调用方式

定义

1.function fn(){}：自定义

2.匿名函数：var fun = function(){};

3.new Function(‘参数1’, ‘参数2’, ‘函数体’)

4.所有的函数都是Function的实例（对象）

5.函数也是对象

调用方式

语法/调用/this指向

1.普通函数：function fn(){}; / fn(); fn.call() / window

2.对象的方法：var o = {sayHi: function(){}} / o.sayHi() / o

3.构造函数：function Star(){}; / var ldh = new Star(); / ldh实例对象，原型对象也是指向实例对象

4.绑定事件函数：btn.onclick = function(){} / 点击即调用 /btn

5.定时器函数：setInterval(function(){}, 1000) / 1秒后自动调用 / window

6.立即执行函数：(function(){})(); / window

js改变函数内this指向的三种方法

1.call()

​ 1）call第一个可以调用函数，第二个可以改变函数内this指向

​ fn.call(o, 1, 2)：fn函数this指向o，传参进fn里

​ 2）call主要作用可以实现继承

​ function Son(..){ Father.call(this,…)};将Father里的this指向Son，继承父类的属性

2.apply():第一个参数改变this指向，第二个参数为数组（伪数组）

​ 1）fn.apply(o, [‘pink’]);

​ 2）apply应用：借助数学内置对象求最大值

3.bind()：不会调用函数，可以改变this指向

​ 1）返回的是原函数改变this之后产生新函数；不需要立即调用，改变this指向用bind

​ 2）应用：改变定时器内部this指向

严格模式

1.给脚本(script标签)开启严格模式：

2.给函数开启严格模式：

​ 1）变量规定：先声明再使用

​ 2）不能随意删除已经声明好的变量

​ 3）this指向问题：1.以前全局作用域中this指向window；严格模式下this是undefined

​ 2.严格模式下，如果构造函数不加new调用，this指向undefined,给它赋值会报错

​ 3.new实例化的构造函数指向创建的对象实例

​ 4.定时器this还是window

​ 5.事件、对象还是指向调用者

​ 4）函数变化：1.严格模式下，函数的参数不能重名

​ 2.函数声明在顶层，不允许在飞函数的代码块内声明函数

高阶函数

1.对其他函数进行操作的函数，它接受函数作为参数或函数作为返回值输出。例如：回调函数

闭包

1.指有权访问另一个函数作用域中变量的函数。变量所在的函数叫闭包。

2.闭包的主要作用：延伸了变量的作用范围

递归

1.定义：函数内部调用自己的函数

原生js的深、浅拷贝

1.浅拷贝只是拷贝一层，更深层次对象级别的只拷贝引用

2.深拷贝拷贝多层，每一级别的数据都会拷贝

3.Object.assign(target, …sources):es6新增方法可以浅拷贝
